# Business-Growth-Analysis
Identified profitability leakage across sales channels and recommended data-driven strategies to improve margins and revenue efficiency.

* Problem
  -------

Businesses often face revenue growth but declining profitability.

* Objective
   ------
Identify key drivers of profit and recommend growth strategies.

* Approach
  -------
Data cleaning
EDA
Profit margin analysis
Segmentation analysis
* Key Insights
  ----------
Highest profit region: 
Best channel: Online
Profit leakage areas: Product 
* Recommendation
   -----------
Optimize low-margin products
Scale high-margin channels
Improve weak customer segments
